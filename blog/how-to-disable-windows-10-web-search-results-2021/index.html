How to disable web results in Windows 10 search (as of 2022)

If this method stops working, please let me know so I can stop it from being included in search results. My email is on my about page.

Windows 10 includes web/Bing search results in its local search. Not great! There used to be a simple toggle to turn off web results. As of January 2022, that simple toggle is gone. Really not great!

This method from a recent reply to an old thread in a random Windows forum is the only way I was able to turn off web results.

Steps to disable web results:

  1. Open the Registry Editor (regedit)
  2. Navigate to "Computer\HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Search"
  3. Create a new "DWORD (32-bit) Value" and name it "BingSearchEnabled" by either right-clicking in the Registry Editor window or by clicking "Edit > New > ..."

  1. Make sure the value data for "BingSearchEnabled" is set to 0
  2. Either (1) restart the File Explorer process or (2) restart your computer
  3. Your Windows search bar should now be free of web/Bing search results
